### measuring_nucleotide_composition.sh
The code used to measure codon frequencies and nucleotide compositions of the open reading frames of Drosophila melanogaster protein coding genes and RepBase-annotated Drosophila Gypsy LTR retrotransposons.

### phylogenetic_analysis.sh
The code used to generate a multiple sequence alignment of Gypsy POL RT domains and infer phylogenetic relationships.

### nucleotide_composition_lineplots_submission.R
R code for making line plots of nucleotide compositions across coding regions and 3'UTRs.

### nucleotide_composition_heatmap_submission.R
R code for making a heatmap of the nucleotide compositions of Gypsy LTR retrotransposon sequences.

### expected_vs_observed_contour.R
R code for making a contour plot, contrasting the distributions of expected and observed adenosine contents of protein coding mRNAs and Gypsy LTR retrotransposons.

### plotting_codon_frequencies.R
R code for plotting the codon frequencies of D. melanogaster protein coding genes and Gypsy LTR retrotransposons.

### RepBase_Drosophila_Gypsy_512ORFs_AA-sequences.GYPSY_17.6_MDG1.curated.fasta
Amino acid sequences of the RepBase-annotated open reading frames of Drosophila Gypsy/Gypsy, 17.6/Gypsy and MDG1/Gypsy elements.

### RepBase_Drosophila_Gypsy_512ORFs_nucleotide-sequences.GYPSY_17.6_MDG1.curated.fasta
Nucleotide sequences of the RepBase-annotated open reading frames from Drosophila Gypsy/Gypsy, 17.6/Gypsy and MDG1/Gypsy elements.

### RepBase_Drosophila_Gypsy.POL_RT.mafft.fasta
Multiple sequence alignment of the 542 RepBase Drosophila Gypsy POL RT domains

### RepBase_Drosophila_Gypsy.POL_RT.mafft.fasta.treefile
Treefile for the phylogenetic analysis of the 542 RepBase Drosophila Gypsy POL RT domains.
Direct output of iqtree using RepBase_Drosophila_Gypsy.POL_RT.mafft.fasta as input.

### RepBase_Drosophila_542.Gypsy_POL-RT_phylogeny.pdf
Phylogenetic tree of the 542 RepBase Drosophila Gypsy POL RT domains.
They are classified into Gypsy, 17.6, 412_mdg1, chimpo, micropia_mdg3, SACCO, osvaldo and outergroup.
These classifications are consistent with the previous study (PMID: 28465726).
